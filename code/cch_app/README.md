# Create Progressive Web apps(PWA) in React
> Installation using npm <br>

 Install node.js via that link <br>
 : https://nodejs.org/ko/

 ```
$ npx create-react-app project_name --template cra-template-pwa
```

## Additional Module Installation
### Axios
**: Promise-based HTTP client for node.js and browser**
```
$ npm install axios
```
### react-router-dom
**: The module used for screen transitions in a Single Page Application (SPA)**
```
$ npm install react-router-dom --save
```

### styled-component
**: A styling framework that facilitates the use of CSS in JavaScript**
```
$ npm install styled-components
```
